Spotify Profiles

Spotify Profiles is an app that lets you take a deep dive into your favorite artists. 

The entire project should be run inside the termProject folder. You can run the project within the editor from spotifyProfiles.py

You'll need the following libraries installed:

cmu_112_graphics.py
pygame
numpy
scikit (sklearn)
requests 
copy
random

Shortcuts:
H: show help screen

shift + R: change color scheme (debug)
